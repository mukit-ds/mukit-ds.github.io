Ongoing Research
Predictive Modeling of League Football Match Outcomes Using Classical Machine Learning
MSc Research Project  ·  East West University, Dhaka, Bangladesh  ·  Sep 2023 — Present
Investigating the application of classical supervised machine learning algorithms — Logistic Regression, Random Forest, Support Vector Machines, Gradient Boosting, and k-NN — to the three-class prediction problem of professional league football outcomes (win / draw / loss). The study bridges statistical physics methodology (stochastic modelling, probability distributions, measurement uncertainty) with rigorous ML pipeline design.
Methodology
  • Structured data collection from publicly available sports databases
  • SMOTE-based class imbalance treatment for equitable model training
  • Temporal train/test splitting to prevent data leakage across seasons
  • Domain-informed feature engineering: rolling window statistics, Elo-style rating systems, form indices
  • Stratified k-fold cross-validation with rigorous statistical evaluation protocols
  • SHAP-based interpretability analysis for model transparency and feature attribution
Preliminary Findings: Ensemble methods (Random Forest, Gradient Boosting) consistently outperform single classifiers; home advantage and recent team form emerge as the most statistically significant predictors. Analysis is ongoing toward final model selection and dissemination.
